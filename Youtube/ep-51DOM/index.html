Frontend development is the development of the graphical user interface of a website, through the use of HTML, CSS, and JavaScript, so that users can view and interact with that website.

  • HTML: The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

  • CSS: Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML

    1. Pure CSS

    2. Bootstrap (Framework)

    3. Tailwind (Framework)

    4. Sass (Preprocessor)

  • JavaScript: JavaScript often abbreviated JS, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. It is use to add functionality in the website.

    1. Vanilla JS

    2. jQuery (Library)

    3. React (Framework)

    4. Angular (Framework)

    5. Vue (Framework)

    6. Typescript (Preprocessor)

innerHTML:Retrieves or sets the HTML content inside an element, including the HTML tags.

element.innerHTML = "Hello"; // The element will render "Hello" in bold.

innerText:Retrieves or sets the visible text of an element, ignoring hidden elements and not preserving the HTML structure.

element.innerText = "Hello World"; // The element will display "Hello World" as plain text (no HTML parsing).

textContent:Retrieves or sets the text content of an element, including the text from hidden elements, but ignores HTML tags and outputs plain text.

element.textContent = "Hello World"; // The element will display "Hello World" as plain text.
