Async / Await

async and await are keywords in JavaScript that simplify working with asynchronous code, especially when dealing with Promises. They make asynchronous code look and behave more like synchronous code, improving readability and making it easier to debug.

1.async Keyword

The async keyword is used to define a function as asynchronous.
An async function always returns a Promise, even if it doesn't explicitly return one.
The returned Promise resolves with the value returned by the async function or rejects if an error is thrown.

2. await Keyword

The await keyword is used inside an async function to pause execution until a Promise is resolved or rejected.
It allows you to write asynchronous code in a synchronous style.
Execution within the function resumes only after the awaited Promise settles (resolves or rejects).

Key Benefits of async and await

#Improved readability: Makes asynchronous code look like synchronous code.
#Error handling: Use try...catch blocks for cleaner error management.
#Sequential operations: Easier to write code that performs actions one after another.
