Try Catch

The try...catch statement in JavaScript is used to handle errors or exceptions gracefully without crashing the program. It allows you to "try" a block of code that might throw an error and "catch" the error to handle it.

# try block: Contains the code that might throw an exception.
# catch block: Executes if an error occurs in the try block. The error is passed as a parameter to the catch block.
