OOP and Factory Functions

In JavaScript, OOP (Object-Oriented Programming) is a programming paradigm based on the concept of objects. Objects are instances of classes, which can contain data (properties) and functions (methods). Here are the key concepts of OOP in JavaScript:

Four Pillars of Object Oriented Programming

  • Encapsulation:-Restricts access to certain details of an object and only exposes necessary parts.
  • Abstraction:-Hiding implementation details and enforcing method contracts (via abstract-like patterns).
  • Inheritance:-Allows a class to inherit properties and methods from another class using extends.
  • Polymorphism:-Overriding parent methods in a subclass.
