'this' keyword in JavaScript

In JavaScript, the this keyword is a special identifier that refers to the execution context in which a function is called. Its value depends on how and where the function is invoked.

Rules for this in JavaScript

Global Context (Outside Any Function): 1.In a browser: this refers to the global window object.
2.In Node.js: this refers to the global object (global), but in ES modules, it's undefined.

Inside an Object Method: this refers to the object the method is called on.

Inside a Regular Function: In non-strict mode, this refers to the global object (window in browsers). In strict mode, this is undefined
