Time Zone: A geographic region where a uniform standard time is used, typically based on an offset from UTC. Examples include GMT and IST.
GMT (Greenwich Mean Time): The time at the Prime Meridian (0Â° longitude), historically the global time standard but now largely replaced by UTC.
UTC (Coordinated Universal Time): The modern global time standard, based on atomic clocks, used as a reference for all time zones.
ISO Standard: The ISO 8601 format provides an internationally accepted way to represent date and time, e.g., 2024-12-14T10:30:00Z.
How Computers Keep Track of Time: Computers use a hardware clock and synchronize system clocks with Network Time Protocol (NTP) servers for accuracy.
Unix Time and Epoch: A time representation counting seconds since January 1, 1970 (Unix Epoch), widely used in programming for its simplicity.
